About Me

 

My name is David Beauchamp. I am a front-end developer with over 6 years of professional experience working across a spectrum of projects and teams.

My specialties include building complex user interfaces from scratch and working hand in hand with designers to bring mockups and concept designs to life. I have collaborated on large projects working in big teams at various levels and have also been the principal developer on smaller initiatives. My main focus is to build useful and engaging user interfaces and experiences. Over the years I have done this in frameworks like React, Angular, WordPress, Shopify, and Jekyll.

Take a look at my GitHub for code examples and personal projects or my Linkedin for my full resume.

What I do

Responsive

Provide an optimal viewing experience—easy reading and navigation with a minimum of resizing, panning, and scrolling—across a wide range of devices.

Modern

Using the latest markup and style techniques for proper semantics and usability. Everything is optimized for browser compatibility and SEO.

Performace

When it comes to building a conversion rate and search engine optimized website, speed is a critical factor. If you don’t have a fast website, people will bounce faster than you can say “conversions.”

CMS

Global Maker Faire's

✻

Domain: eastbay.makerfaire.com, prague.makerfaire.com, sandiego.makerfaire.com

Date: January 2018

Description:

The global Maker Faire program is a collection of 200+ event sites that utilize the WordPress multi site functionality and share a single custom theme. These Maker Faire events needed to have schedule, presenter, and other event related display information easy to use and set up for each non technical website admin.

My role in the project:

I was the solo front-end developer working on this project alongside a couple back-end devlopers. My role was to build a custom WordPress theme based on Photoshop designs. I build a custom page builder system into the WordPress dashboard using Advanced Custom Fields so that each site using the theme could build and move each module for every page. We also built a couple single page Angular apps for displaying and filtering dynamic schedule and presenter information.

Technology used in the project:

WordPress, Angular 1, Advanced Custom Fields, Bootstrap, jQuery, SCSS, MAMP, Grunt
